# Room 0417 🌻

> *"A digital sanctuary for memories, music, and the things that matter most."*

**Room 0417** is a personalized, interactive web experience created for Bella. It serves as a private gallery and a "digital scrapbook" featuring custom-coded interactions, poetry, and music-inspired art.

## 🎞️ The Concept
The project is designed to feel like checking into a private suite. 
- **The Lobby:** A minimalist landing page that welcomes the visitor.
- **The Archives:** Interactive icons (Message, Flowers, Hirono) that reveal hidden notes and "physical" artifacts.
- **The Atmosphere:** Heavily inspired by lo-fi aesthetics, Hirono character art, and the soulful music of Bruno Major.

## 🛠️ Features
- **Paper Texture Rendering:** Custom CSS layers that simulate physical "torn paper" materials for poems and messages.
- **Interactive Vinyl:** A centered, spinning disc for "Nothing" by Bruno Major, featuring synchronized circular SVG text orbit.
- **Responsive "Lobby" UI:** A sleek, top-left status label and an animated scroll-hinting system.
- **Taglish Poetry:** Heartfelt verses that balance traditional Tagalog romanticism with modern "hugot."
